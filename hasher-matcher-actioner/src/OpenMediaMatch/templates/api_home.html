API Details

The HMA API provides endpoints for content hashing, similarity matching, content curation, and system configuration. Please refer to the interactive documentation or the OpenAPI specification for detailed information on each endpoint, request/response schemas, and authentication requirements.

For challenges related to dynamic response structures and guidance for strongly-typed languages, please consult the OPENAPI_INTEGRATION.md guide.
